# 1단계

## 1-1 예약 관리 API

### 예약 도메인

- [ ] 예약은 예약 id, 예약자 이름, 예약 날짜, 예약 시간(24H)로 구성된다. 
  - [ ] 예약 id는 자동으로 생성된다.

### 예약 관리 기능

- [ ] 예약 조회 API
  - [ ] 모든 예약 목록을 조회할 수 있다.
- [ ] 예약 추가 API
  - [ ] 예약자 이름과 시간/날짜를 통해 예약을 추가할 수 있다.
- [ ] 예약 취소 API
  - [ ] 예약 id를 통해 예약을 취소할 수 있다.

## 1-2 데이터베이스 적용

- [ ] 기존 API를 데이터 베이스를 사용하도록 변경한다.
  - [ ] JdbcTemplate을 활용한다.


## 1-3 시간 관리 기능 추가

- [ ] 시간 관리 API를 추가한다.
